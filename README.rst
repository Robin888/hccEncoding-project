Install
-------------------------------------------------------------------------------

``hccEncoding`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install hccEncoding

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade hccEncoding
